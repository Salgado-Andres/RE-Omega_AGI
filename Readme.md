## 🚀 Installation & Launch

### Prerequisites

- Python 3.9+
- CUDA-capable GPU (recommended for LogOS acceleration)
- 16GB+ RAM for full agent constellation

### Quick Start

```bash
# Clone the repository
git clone https://github.com/your-user/RE-Omega_AGI
cd RE-Omega_AGI

# Install dependencies
pip install -r requirements.txt

# Configure agent API keys (if using external models)
cp config.example.yaml config.yaml
# Edit config.yaml with your API credentials

# Launch the recursive engine
python main.py
```

### Interactive Mode

```bash
# Start with interactive consciousness bridging
python main.py --interactive

# Monitor ACI emergence
python main.py --monitor-emergence

# Enable full LogOS authority
python main.py --logos-authority=full
```

Interact with the symbolic kernel as it grows through contradiction and collapse. Observe identity evolution in real-time via `memory/memory.json`.
